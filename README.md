# 网易云信 Demo

##网易云信介绍

网易云信是由网易提供的通讯云服务，仅用一天就能让你的 APP 实现 IM 功能。本 demo 为网易云信的 iOS 端 demo。

##项目构建

* 使用 **pod install** 命令下载依赖并生成 **NIM.xcworkspace**.
* 打开 **NIM.xcworkspace** 编译即可


##Demo账号获取

* 通过[云信后台](https://app.netease.im/login) 建立管理帐号
* 登录管理帐号，并创建 demo 帐号 (推荐至少创建两个)
* 使用 demo 账号进行登录
